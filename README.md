# What is PJM?
PJM is a small Project Manager, it allows administration and movement across projects. Listing them and even performing a regex search.
# Instalation
1. Clone the repo
2. Execute "./install.sh" with a supported shell.
3. Configure your preffered settings.
4. Enjoy.
# Usage
`pjm +<name>`       = creates a new project.
`pjm /[name]`       = changes directory into a project (PROJECTS_DIR if no name).
`pjm -<name>`       = deletes a project.
`pjm l[name[s ...]]`= list select projects, if none is selected, list them all.
`pjm l<pattern>`    = find projects using regex, and list them.
`pjm h`             = prints this message.
